git add

Ajoute des fichiers à la zone de staging en préparation d'un commit.

À quoi sert git add ?

La commande git add permet de sélectionner les fichiers à inclure dans le prochain commit. Elle prépare les modifications pour l'enregistrement dans l'historique du projet.

Exemples d'utilisation

git add fichier.txt
git add .

Ces commandes ajoutent un fichier ou tous les fichiers modifiés à la zone de staging.

Exemple visuel

Astuce

Utilisez git status pour vérifier les fichiers ajoutés à la zone de staging avant de valider avec git commit.
