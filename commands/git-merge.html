git merge

Fusionne une branche dans la branche courante.

À quoi sert git merge ?

La commande git merge fusionne une branche dans la branche courante. Elle permet d'intégrer les modifications d'une autre branche dans votre travail actuel.

Exemples d'utilisation

git merge branche-a-fusionner

Cette commande fusionne la branche branche-a-fusionner dans la branche courante.

Exemple visuel

Astuce

Avant de fusionner, utilisez git fetch pour récupérer les dernières modifications du dépôt distant.
